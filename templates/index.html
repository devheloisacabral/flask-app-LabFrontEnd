Sobre mim

Me chamo Heloisa Cabral, graduanda em Engenharia de Software pela Universidade de Vassouras no Rio de Janeiro.

Comecei os estudos em programação por conta própria e hoje tenho uma trajetória na área de tecnologia de mais de 4 anos.

Possuo experiência profissional como desenvolvedora de sistemas e Designer Gráfico.

Preservo o aprendizado constante, e até hoje sigo prestando prova para bolsas de estudo: Faço parte da turma de Desenvolvimento Back-end da Cubos Academy e da Turma de FullStack StandanderCoders.

Dentro do mercado de trabalho, atuei como dev responsável pelo desenvolvimento de sistemas no TRAINSTATION UNIFAA, onde fui mentorada por programadores Plenos da empresa AKONI.
Após um tempo, atuei como Designer Gráfico no setor de Comercial e Marketing da mesma instituição.

Desenvolvo Landing pages, aplicações na web, sites, interfaces de aplicativos, lojas virtuais, peças gráficas para materiais publicitários, logos e identidades visuais.

Estou no caminho para me especializar na área de Inteligencia Artificial.

Além de programadora e designer, sou tradutora da língua inglesa.

Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.


CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.


JavaScript

JavaScript é uma linguagem de programação interpretada usada no desenvolvimento para descrever funcionalidades e comportamentos da página.


MidJourney e CHATGPT

Inteligência artificial generativa ou IA generativa é um tipo de sistema de IA capaz de gerar texto, imagens ou outras medias em resposta a solicitações em linguagem comum.


React.js

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web


Node.js

O Node.js é um ambiente de execução do código JavaScript do lado servidor (server side), que na prática se reflete na possibilidade de criar aplicações standalone (autossuficientes) em uma máquina servidora, sem a necessidade do navegador.


Python e POO

Django

Django é um framework para desenvolvimento rápido para web, escrito em Python, que utiliza o padrão model-template-view
